Drug Shortage Analysis: Pharma-Pulse

Tools: Python (Pandas, NumPy, Scikit-learn) · Power BI / React JS · Excel · Lucidchart · Google Docs

Target: District and state-level public health procurement officers managing essential generic medicines through GeM and state agencies like TNMSC and MSMSC.

PharmaPulse is a predictive intelligence system that warns public health procurement officers 8–12 weeks before a generic drug shortage occurs. Built across a complete 9-phase BA lifecycle. It aggregates six public data sources (World Bank, IDSP, GeM, CDSCO, FDA) into a hybrid Python risk model and delivers a weekly-refreshed procurement dashboard with an interactive scenario simulator. Every risk score is explainable, every data source is free, and every recommendation is actionable within the standard government procurement window.
